## Summary

Implements task 20 of `002_vibe_coding_plan` by adding the application graph orchestration domain and compiler.

## Changes

- Added `crates/domain/src/application.rs` with:
  - `ApplicationNode`, `Port`, `ApplicationSpec` and `ResourceRef` for model/dataset/stream/secret/device references.
  - `ApplicationSpec::validate` detects cycles, missing nodes and port type mismatches.
  - `ApplicationVersion` computes a canonical digest from the JSON spec on creation and cannot be republished once published.
  - `Application` aggregates versions and tracks `latest_published`.
  - `Binding` resolves node slots to deploy-time `ResourceRef`.
- Added `ApplicationId` to `moqentra-types` and re-exported new types from `crates/domain/src/lib.rs`.
- Marked `APP-001` through `APP-006` as completed in `dev-docs/002_vibe_coding_plan/20_application_orchestration_and_compiler.md`.
